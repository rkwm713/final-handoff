## Quick Reference

### What Each Tracker Is For

| Tracker | Purpose | Team |
|---------|---------|------|
| **Closing** | Track work order closing, as-builts, batch processing, SOX compliance | Closing Team |
| **Drafting** | Track drafting requests, permits, TxDOT RULIS submissions | Drafting Team |
| **EV** | Track Electric Vehicle infrastructure design projects | EV Team |
| **Small Cell** | Track small cell telecommunications projects + SPIDA integration | Small Cell Team |

---

### Common Tasks - Where To Look

| I want to... | Look in... |
|--------------|------------|
| Understand the full project workflow | Project Overview → workflow diagrams and phases |
| Add a new item to a list | User Guide → "How to Make Common Changes" |
| Understand what a field means | User Guide → "Understanding the Fields" |
| Create a filtered view | User Guide → "Views Explained" |
| Know why a date auto-filled | Automation Guide → find the relevant flow |
| Fix a stuck or broken flow | Automation Guide → "Troubleshooting" |
| Refresh Excel data from SharePoint | Excel Info → "Refreshing Data" |
| Add a column to Excel reports | Excel Info → "Making Changes" |
| Understand the M Code | Excel Info → "M Code Reference" |
| Get AI help for a specific problem | AI Prompts → find a relevant prompt template |
| Learn Power Automate expressions | AI Prompts → "Write a Power Automate expression" |

### Key Automations Summary

| Tracker | What Runs Automatically |
|---------|------------------------|
| **Closing** | Days In Status calculation, SOX Timer, FCC/Storeroom date tracking, Blocker dates |
| **Drafting** | Date stamps on status change, RULIS info alerts via Teams |
| **EV** | Status change logging to Excel with duration calculations |
| **Small Cell** | Status End Date recording for each workflow stage |
